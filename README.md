# 🐍 Python do Zero com Copiloto IA

Bem-vindo ao repositório **Python do Zero com Copiloto IA**! Este projeto é um guia passo a passo para aprender Python desde o básico, usando inteligência artificial como assistente.

## 📚 Sobre o Projeto

Este repositório contém materiais e códigos desenvolvidos durante o curso "Python do Zero com Copiloto IA" pela **KENSEICYBERSECURITY ACADEMY**. Cada semana aborda conceitos específicos da linguagem Python, com exemplos práticos, explicações didáticas e exercícios para iniciantes.

O objetivo é tornar o aprendizado de programação acessível, divertido e progressivo, utilizando ferramentas de IA para acelerar o processo.

## 📁 Estrutura do Repositório

- **`semana 2/`**: Pasta da Semana 2 - Listas, Dicionários, Condicionais e Loops.
  - Exemplos de estruturas de dados, tomadas de decisão e repetições.
  - Arquivos: `ola.py`, `variaveis.py`, `listas_dicts.py`, `condicionais.py`, `loops.py`.
  - `README_semana2.md` detalhado com explicações para iniciantes.

*(Outras semanas serão adicionadas conforme o progresso do curso.)*

## 🚀 Como Usar

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/NaiaraRodrigues-naih/PYTHON-DO-ZERO-COM-COPILOTO-IA.git
   ```

2. **Navegue para a semana desejada** (ex: `cd semana\ 2`).

3. **Execute os arquivos Python** (certifique-se de ter Python instalado):
   ```bash
   python listas_dicts.py
   ```

4. **Leia o README.md** de cada semana para explicações detalhadas.

## 🛠️ Pré-requisitos

- **Python 3.x** instalado (recomendado 3.8 ou superior).
- Um editor de código como VS Code, PyCharm ou similar.
- Conhecimento básico de terminal/comando.
